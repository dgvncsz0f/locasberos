LOCASBEROS
==========

A apache/nginx authentication module.


Configuration Directives (sketch)
---------------------------------

* CasEndpoint
* CasLoginURL
* CasServiceValidateURL

* CasService
* CasRenew

* CasCookieName
* CasCookieTimeout
* CasCookiePath

Dependencies
------------

* libcurl >=7.25
* libxml2 >=2.7.8

Test Dependencies
-----------------

* libunittest++ >=1.4
